These are the incremental builds leading up to JMesa 2.3. The naming convention is jmesa-version_year.month.day.zip The version represents the version that I am working towards, and then the rest includes the date the build was made up.

Be sure to visit the wiki for a complete listing of the JMesa 2.3 feature roadmap and how much of it is done. I will also include the backwards compatible of each feature...although I do not expect any of the new features to effect the rest of the library. If anything JMesa may depend on the JQuery JavaScript library for the advanced features but not for the existing functionality.
